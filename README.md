# Koala-Site

Aplikasi yang saya buat untuk memberikan letter kepada orang yang saya sayang karena dia mau utbk dan saya bosan kasih letter dalam bentuk whatsapp terus. 

The project is built using React.js with JSX for component-based UI development. Styling is implemented using standard CSS focus on responsive design across devices. For animations and micro-interactions I utilizes Framer Motion for this website

### 🧩 Tech Stack:
* Frontend: React.js (JSX)
* Animation: Framer Motion
* Version Control: Git & GitHub
* Deployment: Vercel

# Getting started
