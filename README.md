# Koala-Site

Aplikasi yang saya buat untuk memberikan letter kepada orang yang saya sayang karena dia mau utbk dan saya bosan kasih letter dalam bentuk whatsapp terus. 

The project is built using React.js with JSX for component-based UI development. Styling is implemented using standard CSS (or utility-based styling depending on structure), with a focus on responsive design across devices. For animations and micro-interactions, the project utilizes Framer Motion, enabling smooth transitions such as element fade-ins, scaling effects, and interactive icon animations.

### 🧩 Tech Stack:
Frontend: React.js (JSX)
Animation: Framer Motion
Styling: CSS / (optional: Tailwind CSS if used in your project)
Version Control: Git & GitHub
Deployment: Vercel
