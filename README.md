# Koala-Site

Aplikasi yang saya buat untuk memberikan letter kepada pacar saya karena dia mau utbk dan saya bosan kasih letter dalam bentuk whatsapp terus. Aplikasi ini dibuat menggunakan jsx dan framer motion sebagai animasi pada website ini

# Getting started

Run, the develompent server
```
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open local host http://localhost:5173 to see the result
